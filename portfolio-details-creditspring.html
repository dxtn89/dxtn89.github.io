Creditspring

Project information

  • Project URL: creditspring.co.uk

Creditspring offers a new model for personal finance and on-demand loans. Members can get a couple of loans per year, based on their current needs. The process of lending money is fully transparent and based on membership so there are no hidden costs. The core of the system is the admin panel which has been built in Ruby on Rails and allows Creditspring agents to manage usersâ€™ membership as well as payment processing for all loans.

While our team was dedicated to the development process, we worked on implementation of following features: multi-user configurations, admin panel with user, role & access management, platform integrations with APIs, and development of some modules for the recruiting CRM. The solution is currently available for the US market.

Technologies : Ruby on Rails, PostgreSQL, Sidekiq, Redis, jQuery, Stimulus.js, Heroku, RSpec
